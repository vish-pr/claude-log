# Claude Code Log Viewer

Single-file HTML viewer for Claude Code JSONL conversation logs. Drag-and-drop a log file to browse messages with syntax highlighting.

**Completely local** — no data is sent anywhere. Everything runs in your browser.

## Usage

**Option A — Local file**

Download `index.html` and open it in a browser:

```
file:///path/to/index.html
```

**Option B — Hosted**

Visit [vishpr.com/claude-log](https://vishpr.com/claude-log)
